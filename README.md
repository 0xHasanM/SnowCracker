# SnowCracker
Snowcracker used to crack Stegsnow passwords

# Install :

1. `pip install futures`

2. `git clone https://github.com/0xMohammed/SnowCracker.git`

# Usage :

`python3 snowcracker.py -c Y -f flag.txt -w wordlist `
